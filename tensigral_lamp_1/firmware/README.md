# Tensigral lamp firmware demo

For operation on an STM32F042 with the tensigral lamp firmware, using SK6812-RGBW LEDs.

# Prerequisites

sudo apt-get install gcc-arm-none-eabi libnewlib-arm-none-eabi openocd
